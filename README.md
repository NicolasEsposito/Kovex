# Kovex

Sitio web de Kovex construido con **Vite + TypeScript + React**.

## Estructura

El proyecto vive en la carpeta [`kovex-web/`](kovex-web/).

## Desarrollo

```bash
cd kovex-web
npm install      # instalar dependencias (solo la primera vez)
npm run dev      # levantar el servidor de desarrollo
npm run build    # generar la versión de producción en dist/
```
